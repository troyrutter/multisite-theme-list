Display table of multisite sites with wordpress theme and version.

== Description ==

This plugin presents a simple table in the Network Admin > Themes section that lists each multisite and what theme/version is currently active. 
At the end of the listing a summary is given listing the number of sites active for each theme in use.


== Installation ==

In WordPress:

1. Go to Plugins > Add New > search for `multisite-theme-list`
1. Press "Install Now" for the AddToAny plugin
1. Press "Activate Plugin"

Manual installation:

1. Upload the `multisite-theme-list` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the "Plugins" menu in WordPress
